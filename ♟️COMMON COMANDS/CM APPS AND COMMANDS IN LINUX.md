==Change keyboard language==

| Layout code | Description                               |
| ----------- | ----------------------------------------- |
| us          | English (US QWERTY)                       |
| es          | Spanish (Spain)                           |
| latam       | Latin American Spanish (Colombia, others) |
| fr          | French AZERTY                             |
| de          | German QWERTZ                             |
| ru          | Russian Cyrillic                          |
| colemak     | Alternative layout                        |
| dvorak      | Alternative layout                        |
